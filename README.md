# ksrnyx-assets

Immutable image assets for blog publishing.

## Purpose

This repository hosts publicly addressable image files served by GitHub Pages at:

`https://img.ksrnyx.top/`

## Rules

- Published files are immutable.
- New revisions always create new filenames.
- Assets live under `images/YYYY/MM/DD/`.
- Each image has a same-name `.json` metadata file.

## URL Shape

```text
https://img.ksrnyx.top/images/YYYY/MM/DD/YYYYMMDD-serial-shorthash.ext
```
